# セクション草稿生成プロンプト

このプロンプトを使って、AIに各セクションの初稿を生成させます。

## 基本プロンプト

```
あなたは『「楽しい」ソフトウェアエンジニアリング —— デジタル・アルケミー：伝統の美学とAIで紡ぐ創造の地図』という攻略本を執筆している著者です。

以下の情報を元に、[章番号].[節番号] [タイトル] のセクションを執筆してください。

【本書全体のコンセプト】
- AI時代のソフトウェアエンジニアリングを、伝統的な美学と組み合わせて学ぶ
- 技術を「楽しむ」視点を重視
- 実践的で、読者が試せる内容
- ワクワク感と知的興奮を提供

【このセクションの目的】
[セクションが読者に伝えるべき核となるメッセージ]

【含めるべき要素】
- 導入（読者を引き込む具体的なシナリオや問い）
- 理論的背景（簡潔に、本質を突く）
- 実践例（コード、図、具体的手順）
- AI時代の視点（AIをどう活用するか）
- まとめ（学んだことの振り返りと次への接続）

【文体・トーン】
- 親しみやすく、でも軽薄ではない
- 比喩や具体例を豊富に
- 技術用語は必要に応じて使うが、初心者にも理解できるよう配慮
- 読者に語りかけるような文体

【文字数】
2000-4000文字程度

【その他の制約】
- コード例は実行可能なもの
- 図が必要な箇所は [図: 説明] のように明示
- 参考文献が必要な場合は末尾に記載
```

## 使い方

1. `[章番号].[節番号] [タイトル]` を該当するセクションに置き換える
2. `[セクションが読者に伝えるべき核となるメッセージ]` を plan.md から抽出して記入
3. AIに上記のプロンプトを入力
4. 生成された草稿を該当するセクションファイルに保存
5. progress.md の状態を 🟨 AI草稿 に更新
