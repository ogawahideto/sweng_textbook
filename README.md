## 執筆ワークフロー

### 📝 基本的な流れ

```
1. 次のタスクを決定
   ↓
2. AIでセクションの草稿を生成
   ↓
3. 人間が修正・追記
   ↓
4. AIでレビュー・改善提案
   ↓
5. 再修正
   ↓
6. 完成したら次のタスクへ
```

## クイックスタート

初めて使う場合は `.ai/QUICKSTART.md` を参照してください。

## ステップバイステップガイド

### ステップ1: 次のタスクを決定する

AIに以下のように指示：

```
@.ai/workflows/plan-next.md のワークフローに従って、
次に取り組むべきタスクを提案してください。

@progress.md
```

提案を検討し、自分で次のタスクを決定後、`progress.md` を更新

### ステップ2: AIでセクションの草稿を生成

AIに以下のように指示：

```
@.ai/workflows/draft.md のワークフローに従って、
[章番号].[節番号] [タイトル] の草稿を生成してください。

セクション目的:
@plan.md から該当部分

スタイル: @style-guide.md に準拠
構造: @templates/section-template.md に従う
```

生成された草稿を `chapters/partX/chapterYY/X-Y.md` に保存し、
`progress.md` の状態を 🟨 AI草稿 に更新

### ステップ3: 人間が修正・追記

1. 生成された草稿を読む
2. 以下の観点で修正・追記
   - 技術的な正確性の確認
   - 自分の経験や視点の追加
   - 具体例やコードの改善
   - 文体の調整
   - 図の追加（必要に応じて）
3. 修正内容を保存
4. `progress.md` の該当セクションの状態を 🟦 人間修正中 に更新

### ステップ4: AIでレビュー・改善提案

AIに以下のように指示：

```
@.ai/workflows/review.md のワークフローに従って、
以下のセクションをレビューしてください。

セクション: [章番号].[節番号] [タイトル]

[修正後のセクション全文を貼り付け]
```

フィードバックを受け取り、検討

### ステップ5: 再修正

1. AIの提案の中から採用するものを選択
2. さらに自分の視点で改善
3. 最終版を保存
4. `progress.md` の該当セクションの状態を 🟩 レビュー待ち に更新

### ステップ6: 完成の判断と次へ

1. セクションが満足できる品質か確認
2. OKなら `progress.md` の状態を ✅ 完成 に更新
3. まだ改善の余地があれば、ステップ3-5を繰り返す
4. 完成したら、ステップ1に戻って次のタスクへ

## 執筆のヒント

### AIとの協働のコツ

1. **AIに任せるべきこと**
   - 構造の整った初稿の生成
   - 網羅的な情報の列挙
   - 既存の知識の整理
   - 文章の推敲

2. **人間が担うべきこと**
   - 独自の視点や経験の追加
   - 具体例の選定
   - 読者への共感
   - 最終的な判断

3. **効果的な指示のコツ**
   - ワークフローファイル（@.ai/workflows/）を参照させる
   - 必要なコンテキスト（@plan.md など）を明示
   - 具体的な期待値を伝える
   - 必要に応じて例を示す

### 執筆時の注意点

1. **一貫性を保つ**
   - 用語の統一
   - 文体の統一
   - 構成の統一

2. **読者視点を忘れない**
   - 初心者にも理解できるか
   - 実践的な内容か
   - ワクワク感があるか

3. **品質と進捗のバランス**
   - 完璧を求めすぎない
   - でも妥協もしない
   - 定期的に全体を見直す

## 進捗管理のポイント

### 定期的にすること

1. **毎日**
   - 少しでも執筆を進める
   - `progress.md` を更新

2. **週に1回**
   - 全体の進捗を確認
   - バランスを調整（特定の章だけ進みすぎていないか）
   - 次週の計画を立てる

3. **月に1回**
   - 既存のセクションを読み返す
   - 一貫性をチェック
   - 全体の流れを確認

### モチベーション維持

1. **小さな成功を祝う**
   - 1セクション完成したら自分を褒める
   - 進捗を可視化して達成感を得る

2. **柔軟に進める**
   - 気分が乗らないセクションは後回し
   - 今書きたいセクションを優先してもOK

3. **仲間を作る**
   - 進捗をSNSでシェア
   - フィードバックをもらう

## よくある質問

### Q: どのセクションから始めるべき？

A: 以下の3つの選択肢があります：
1. **順序通り（プロローグから）**: 論理的で、読者の視点で書ける
2. **得意なセクションから**: モチベーション維持に有効
3. **重要なセクションから**: コア部分を先に固める

自分に合った方法を選んでください。

### Q: AIが生成した内容をどこまで信頼していい？

A: AIは優秀なアシスタントですが、以下の点に注意：
- 技術的な正確性は必ず自分で確認
- 最新情報は自分で調べて補完
- 独自の視点や経験は人間が追加
- 最終的な判断は人間が行う

### Q: セクションの品質をどう判断する？

A: 以下のチェックリストを使用：
- [ ] 技術的に正確か
- [ ] 初心者にも理解できるか
- [ ] 実践的な内容か
- [ ] ワクワク感があるか
- [ ] 前後のセクションとつながっているか
- [ ] 本書のコンセプトに合っているか

### Q: 行き詰まったら？

A: 以下を試してください：
1. 別のセクションに取り組む
2. `prompts/next-task.md` で新しい視点を得る
3. 既存のセクションを読み返して刺激を得る
4. 一度離れて休憩する
5. 誰かにフィードバックをもらう
