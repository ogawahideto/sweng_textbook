# Estimate Pages

原稿ファイルの分量から書籍としてのページ数を概算します。

## 使い方

```
/estimate-pages              # 全体のページ数を計算
/estimate-pages chapter01    # 特定の章のみ
/estimate-pages --detailed   # セクション単位の詳細表示
```

## 計算基準

### デフォルト設定（A5判 技術書）

| 項目 | 値 |
|------|-----|
| 本文1ページあたり文字数 | 800文字 |
| コードブロック1行あたり本文換算 | 50文字 |
| 図・表1点あたり本文換算 | 400文字（0.5ページ） |

### オプション: 判型の指定

```
/estimate-pages --format b5   # B5判（1200文字/ページ）
/estimate-pages --format a4   # A4判（1500文字/ページ）
```

| 判型 | 文字/ページ |
|------|------------|
| A5（デフォルト） | 800 |
| B5 | 1200 |
| A4 | 1500 |

## 実行手順

### Step 1: 原稿ファイルを収集

`chapters/` 以下の `.md` ファイルを再帰的に収集。

除外するファイル:
- `*-proposal.md`（企画書）
- `README.md`

### Step 2: 各ファイルを解析

各ファイルに対して以下をカウント:

1. **本文文字数**: マークダウン記法を除いた純粋なテキスト
   - 見出し（`#`）、強調（`**`、`*`）、リンク記法は除去
   - 空行・空白は除外

2. **コードブロック行数**: ` ``` ` で囲まれた部分の行数
   - コードは本文より1行あたりの情報量が少ない
   - 行数 × 50文字として本文に換算

3. **図・表の数**:
   - `![` で始まる行（画像）
   - `|` を含む行が3行以上連続（テーブル）
   - SVGファイル参照
   - 1点あたり0.5ページ換算

### Step 3: ページ数を計算

```
総文字数 = 本文文字数 + (コード行数 × 50) + (図表数 × 400)
ページ数 = 総文字数 ÷ 文字数/ページ
```

余白や章扉ページを考慮して **1.1倍** の補正を加える。

### Step 4: 結果を表示

## 実装のヒント

### 文字数カウント（Bash）

```bash
# 本文文字数（コードブロック除外）
cat file.md | sed '/^```/,/^```/d' | sed 's/[#*`\[\]()]//g' | tr -d '\n ' | wc -m

# コード行数
cat file.md | sed -n '/^```/,/^```/p' | grep -v '^```' | wc -l

# 画像数
grep -c '!\[' file.md

# テーブル行数（3行以上連続する | 含む行をテーブルとしてカウント）
# 簡易的には | を含む行数を3で割って図表数とする
grep -c '|' file.md
```

### PowerShell版

```powershell
# Markdownファイルの解析
$content = Get-Content -Raw $file
$codeBlocks = [regex]::Matches($content, '```[\s\S]*?```')
$codeLines = ($codeBlocks | ForEach-Object { ($_.Value -split "`n").Count - 2 } | Measure-Object -Sum).Sum
$textOnly = $content -replace '```[\s\S]*?```', '' -replace '[#*`\[\]()!]', '' -replace '\s', ''
$charCount = $textOnly.Length
$imageCount = ([regex]::Matches($content, '!\[')).Count
$tableRows = ([regex]::Matches($content, '^\|.+\|$', 'Multiline')).Count
```

## 参考: 一般的な技術書のページ数

| 分類 | ページ数目安 |
|------|------------|
| 入門書・薄い本 | 100〜150 |
| 標準的な技術書 | 200〜350 |
| 網羅的な解説書 | 400〜600 |
| 辞典・リファレンス | 600〜1000+ |
