# Session Log Update

セッションログを更新します。

## 手順

1. 最新のセッションログファイルを確認する（`.ai/sessions/`）
2. 前回のログ日付と今日の日付を比較
3. 未記録の日がある場合、gitログから活動内容を抽出
4. 新しいセッションログを作成または更新

## 実行内容

### Step 1: 既存ログの確認

```bash
ls -la .ai/sessions/
```

### Step 2: 今日の日付を取得

今日: $CURRENT_DATE

### Step 3: Gitログから活動を抽出

```bash
git log --oneline --since="yesterday" --until="now"
```

### Step 4: セッションログの作成

以下のフォーマットで `.ai/sessions/YYYY-MM-DD.md` を作成:

```markdown
# セッションログ - YYYY-MM-DD

## 概要
[1-2行で今日の活動を要約]

## 主な活動
- [活動1]
- [活動2]
- [活動3]

## 作成・更新したファイル
- [ファイル1]
- [ファイル2]

## Gitコミット
- [コミットメッセージ1]
- [コミットメッセージ2]

## 次のステップ
- [次にやること]
```

### Step 5: 結果を報告

作成または更新したログファイルの内容を表示し、コミットする。
